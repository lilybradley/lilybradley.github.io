How I learned to code: A work in progress

Step 1: Complete Codecademy's HTML and CSS course. 20-40 hours.

Step 2: Study websites' HTML & CSS by:

  1. Enabling developer tools in your web browser.
    On Safari (Mac), go to Preferences (command + ",") > Advanced > click "Show Develop menu in menu bar".
    Now it appears on the menu between "Bookmarks" and "Window".
  2. Visiting websites you like, and click "Show Web Inspector" from the developer menu. 2-4 hours.

Step 3: Try to create a website.
Blogs are popular personal websites. WordPress provides free and paid templates and is supposed to be easy to use. I got stuck trying to connect it with my domain name and the domain name server and other nonesense. Very frustrating; not at all covered by the Codecademy course; and not relevant to learning to code. I ended up finding github pages easiest to set up with the caveat being that I happily accepted the .github.io url (to not deal with aforementioned hassle). 2-200 hrs.

Step 4: Check out GitHub. 10-20 hrs.

  • Sigh. They make you learn a lot of technical jargon upfront. When I have time I will create a cheat sheet. A few important ones:
    • "git" is "save as" - but more flexible to allow for multiple editors/versions
    • "pull request" is a "please use this version with my changes"
    • "commit" is "I accepted your changes and saved"
    • "clone" is "download an offline version"
  • I like GitHub because it provides ample opportunity for imitation (highest form of flattery, right?). Just make sure you "cite your source" - give credit, read the copyright and licensing information. Note that stuff developed by the US government by nature is in the public domain and therefore is not copyrightable.

Step 5. Set up GitHub.

  • Sign up for an account.
  • Set up your GitHub page by following these directions.
  • Download GitHub Desktop. For Mac. Did you set up your own GitHub page yet? If so, "clone" (i.e., download an offline version) to your desktop.
  • Turn your text editor into an HTML editor: YouTube instructional video for Mac
